# Arithmetic_Logic_Unit
Toate fisierele pentru un ALU functional in Verilog + documentatia.
In cn.zip se afla harta de Minecraft pe care am realizat-o pentru sumator, downnload-abila pentru a putea fi vazuta/jucata de oricine.
Structura proiectului cuprinde 1 testbench mare pentru tot ALU si 4 mici(1 pentru fiecare operatie) alaturi de module verilog pentru fiecare operatie si pentru Control Unit sin ALU_top si registru simplu, dar si documentatia in Word si prezentarea in Powerpoint.
